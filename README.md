# AI-администратор для сервисного бизнеса — демо

Event-driven AI-ассистент: принимает входящие сообщения, понимает намерение через LLM и выполняет действие — отвечает клиенту, записывает на услугу, ведёт лид в CRM, уведомляет администратора.

**Как работает:**
`сообщение → webhook → n8n → LLM-классификация намерения (OpenAI) → маршрутизация → ответ / CRM / уведомление`

**Стек:** Python · n8n · OpenAI API (LLM) · Telegram Bot API (aiogram) · Google Sheets · webhooks · CRM

**Живое демо:** https://msaidamir98-spec.github.io/ai-admin-demo/ (или открой `index.html` локально)

> Данные в демо обезличены. Боевые внедрения делаю под ключ: бриф → сборка → интеграции → production → обучение.
